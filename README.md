# Linux-Lernplattform

Strukturierte Lerninhalte und Übungsaufgaben für Linux-Einsteiger.

## Kursaufbau
* `lesson01/`: Erste Schritte im Terminal (Übungsdateien).
* `lesson02/`: Dateisystem und Berechtigungen.
* `lesson03/`: Prozesse und Netzwerk.
* `exams/`: Prüfungsaufgaben und Lösungen zur Wissenskontrolle.
